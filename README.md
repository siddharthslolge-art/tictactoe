# Tic Tac Toe

A simple Tic Tac Toe game built using Python.

## Features

- Two-player gameplay
- Detects winner automatically
- Detects draw situations
- Simple and easy-to-use interface

## Technologies Used

- Python
- Tkinter / Pygame (choose whichever you used)

## How to Run

1. Install Python
2. Download the project files
3. Open terminal in the project folder
4. Run:

```bash
python tic_tac_toe.py
```

## Game Rules

- Players take turns placing X and O.
- The first player to get three symbols in a row, column, or diagonal wins.
- If all cells are filled and nobody wins, the game ends in a draw.
